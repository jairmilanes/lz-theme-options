Theme Options Fields

Here is a list of the currently available fields, please keep in mind that, some of this fields may still have bugs, but most of them are ready to use, updates will come to fix all the current bugs.

Help is always welcome, please use the Github Issue System to register and discuss bugs, solutions and submit pull requests.

Text field

Simple input type text

'field_name' => array(
    'type' => 'text', // Field type
    'options' => array( // Field options
        'label' => __('Field title','theme_name'), // Field Label
        'required' => false, // Field is required?
        'placeholder' => 'Placeholder text' // Fields placeholder string
        'max_length' 	=> 25,
				'min_length' 	=> 3,
    ),
    'description' => __('Field description','theme_name')
)

Ajax File field

Ajax upload field

'field_name' => array(
    'type' => 'ajaxFile', // Field type
    'options' => array( // Field options
        'label' => __('Field title','theme_name'), // Field Label
        'required' => false, // Field is required?
        'placeholder' => 'Placeholder text' // Fields placeholder string
        'type' => 'image', // Upload type ( image[jpg,png,gif,ico], archive[zip, rar], document[doc, pdf], all, custom[[mime_type]])
        'max_size' => 2000, // Max upload size in bytes
        'width' => '1024', // Max image with ( only for image uploads )
        'height = '768', // Max image height ( only for image uploads )
        'min_width' => '600', // Min image width ( only for image uploads )
        'min_height' => 480, // Min image height ( only for image uploads )
        'multiple' => false // Should allow multiple uploads if so how many ( *Under development )
    ),
    'description' => __('Field description','theme_name')
)

If you choose to use a custom upload type remember to pass a "array of mime_types", even if is only one mime_type you want to allow.

Texture selector field

Allows to choose from a list of textures

'field_name' => array(
    'type' => 'textureSelector', // Field type
    'options' => array( // Field options
        'label' => __('Field title','theme_name'), // Field Label
        'required' => false, // Field is required?
        'option_size' => 'large', // The size of each color square (small, medium, large)
        'choices'	=> array(
          'texture-1' => 'img/textures/texture-1.jpg', // Name and path to the texture file
          'texture-2' => 'img/textures/texture-2.png',
          'texture-3' => 'img/textures/texture-3.gif',
          'texture-4' => 'img/textures/texture-4.jpg',
          'texture-5' => 'img/textures/texture-5.png',
          'texture-6' => 'img/textures/texture-6.gif',
          'texture-7' => 'img/textures/texture-7.jpg'
        )
    ),
    'description' => __('Field description','theme_name')
)

The "choices" parameter can also be a path, relative to you theme root folder, it will try and load textures from that folder.

Google font selector field

Allows to choose from a list of all available Google Fonts, a few filters are available to narrow down the list if needed.

'field_name' => array(
    'type' => 'googleFont', // Field type
    'options' => array( // Field options
        'label' => __('Field title','theme_name'), // Field Label
        'required' => false, // Field is required?
        'api-key' => 'AIzaSyBTY43wLYJEOnJY7IiChTn8-JCZ-TsaKRB', // Google fonts api key
        'sort' => 'popularity', // alpha, date, popularity, style & trending
        'variants' => array( 'regular', 200, 600), // If the font does not have one of the sizes it will load the available ones
        'subsets' => array('latin', 'greek'), // Arabic, Cyrillic, Cyrillic Extended, Devanagari, Greek, Greek Extended, Khmer, Latin, Latin Extended, Telugu, Vietnamese
        'category' => array('serif','sans-serif'), // serif, sans-serif,
        'limit' => 10 // Limit the number of fonts loaded by group
    ),
    'description' => __('Field description','theme_name')
)

This setup will load a 10 fonts for each of the categories, including each subset & variants sorted by popularity."

Country selector field

Displays a select box with all the available countries to choose from

'field_name' => array(
    'type' => 'countrySelector', // Field type
    'options' => array( // Field options
        'label' => __('Field title','theme_name'), // Field Label
        'required' => false, // Field is required?
        'limit' => array('US','BR'), // This limits the list of countries to the ones in the array
        'exclude' => array('US','BR') // This will eclude this countries from the list
    ),
    'description' => __('Field description','theme_name')
)

Region selector field

Displays a select box with all the available regions to choose from given a country code

'field_name' => array(
    'type' => 'regionSelector', // Field type
    'options' => array( // Field options
        'label' => __('Field title','theme_name'), // Field Label
        'required' => false, // Field is required?
        'country_field' => 'Country field name', // This will make this field change according to the selected country
        'country' => '', // Default region to load cities from ( name or id )
    ),
    'default' => 'Default region name', // selected city id, only if used region from options
    'description' => __('Field description','theme_name')
)
