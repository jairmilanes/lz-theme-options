Getting started

Integrating the Theme Options plugin with your template is easy, the first thing to do is to install the plugin, you can clone it from Github

git clone https://github.com/layoutzweb/lz-theme-options.git

or download a installable copy of Lz Theme Options, witch can be installed using the oc-admin plugins area, clique here to download

Don´t forget to "intall" the plugin after uploading to your server.

Creating the options file

With the plugin installed you can now start creating your configuration options, to do that first create a "options.php" file in the root directory of your theme, located at:

path/to/root/dir/oc-content/themes/your-theme-name

Now inside this file create a function that the name is formed by your theme slug name followed by "_get_theme_options", this function must return a array, like the exemple below:

function mythemename_get_theme_options(){
  return array();
}

We are now ready to start creating fields.

Creating fields

Fields are created using an array notation, nesting arrays to group fields, currently the Lz Theme Options plugin cames with a variety of fields, check out all the available fields and their basic configuration markup in the Fields section.

Here is a complete options file, you can keep extending the options ti fit your needs:

function mythemename_get_theme_options(){
	return array(
		'first_group' => array(
			'title'  => __('First group title','theme_name'),
			'fields' => array(
				'first_field' => array(
					'type' => 'text',
					'options' => array(
						'label' => __('Field title','theme_name'),
						'required' => false,
						'placeholder' => 'Placeholder text',
						'max_length' 	=> 25,
						'min_length' 	=> 3
					),
					'description' => __('Field description','theme_name')
				),
				'second_field' => array(
					'type' => 'textarea',
					'options' => array(
						'label' => __('Field title','theme_name'),
						'required' => false,
						'placeholder' => 'Placeholder text',
						'cols' => 60,
						'rows' => 6
					),
					'description' => __('Field description','theme_name')
				),
				'third_field' => array(
					'type' => 'select',
					'options' => array(
						'label' => __('Field title','theme_name'),
						'required' => false,
						'choices' => array(
							'key' => 'value',
							'key2' => 'value2',
							'key3' => 'value3'
						)
					),
					'description' => __('Field description','theme_name')
				)
			)
		),
		'second_group' => array(
			'title'  => __('Second group title','theme_name'),
			'fields' => array(
				'first_field' => array(
					'type' => 'checkbox',
					'options' => array(
						'label' => __('Field title','theme_name'),
						'required' => false,
						'choices' => array(
							'key' => 'value',
							'key' => 'value'
						)
					),
					'description' => __('Field description','theme_name')
				),
				'second_field' => array(
					'type' => 'toggleSwitch',
					'options' => array(
						'label' => __('Field title','theme_name')
					),
					'description' => __('Field description','theme_name')
				),
				'third_field' => array(
					'type' => 'ajaxFile',
					'options' => array(
						'label' => __('Field title','theme_name'),
						'required' => false,
						'placeholder' => 'Placeholder text',
						'type' => 'image',
						'max_size' => 2000,
						'width' => 1024,
						'height' => 768,
						'min_width' => 600,
						'min_height' => 480,
						'multiple' => false
					),
					'description' => __('Field description','theme_name')
				)
			)
		)
	);
}

Checking the form

With some some options created you can now check how is your settings looking in the backend, look for the "Theme Options" on your oc-admin appearance menu.

Once there you will see the preview window, with a big square button on the left side, this is the settings menu trigger.

Clique the button to open the settings you have just created above.

Retrieving values

To retrieve the settings values you can use one of the functions below, choose the best one for each situation:

lzto_getOptionsByGroupName( $group_name );

"lzto_getOptionsByGroupName" returns an array with the field name and value for each field on that group.

lzto_getOption( $group_name, $field_name );

"lzto_getOption" returns the value of the field, maybe a string, array, number, depends on the field type.

lzto_getSwitchOption( $group_name, $field_name );

"lzto_getSwitchOption" returns true or false for a "toggle switch" type of field.

lzto_getGoogleFont( $group_name, $field_name );

"lzto_getGoogleFont" returns an array with the name and url of the chosen font, you can then use the url to enqueue the google font and the name to overwrite you css.

That's is it!
